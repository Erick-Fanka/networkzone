# 📁 DHCP Server — Laboratório

Laboratório de configuração de servidor DHCP com `isc-dhcp-server` no Ubuntu Server em ambiente virtualizado com VirtualBox.

## Estrutura

```
dhcp/
├── configs/
│   └── dhcpd.conf        # Arquivo de configuração do servidor DHCP
└── docs/
    ├── dhcp-conceitos.md  # Explicação dos conceitos do protocolo DHCP
    └── dhcp-passo-a-passo.md  # Guia prático de configuração do laboratório
```

## Tecnologias

- Ubuntu Server 22.04
- Linux Mint 21
- VirtualBox (Internal Network)
- `isc-dhcp-server`
